# Animations

In orderd to start an animation you need to add:

ANIMATION | DURATION | ANIMATION TYPE | CUBIC BEZIER (optional)

Example: <h1 class="animate d1s fadein easeInOutQuad">example</h1>



Other Classes

Class Alternative Class Purpose
.animate Required to start the animation
.invisible .hide Setting element's opacity to 0, good for hiding elements when animation starts
.infinite .loop Runs the animation forever
.reverse Specifies whether an animation should play in reverse direction
.alternate The animation will be played as normal every odd time (1,3,5,etc..) and in reverse direction every even time (2,4,6,etc...)
.alternate-reverse The animation will be played in reverse direction every odd time (1,3,5,etc..) and in a normal direction every even time (2,4,6,etc...)


Animation Duration

now you can use also halfs: .d4s5 by adding a 5 after the s
From class To class Description
from .d1ms to .d9ms animations play in milliseconds
from .d1s to .d15s animations play in seconds
half .d9s5   To use half seconds, add a 5 at the end of s


Animation Delay

now you can use also halfs: .lag-4s5 by adding a 5 after the s
From class To class Description
from .lag-1ms to .lag-9ms animations play in milliseconds
from .lag-1s to .lag-15s animations play in seconds
half .lag-9s5   To use half seconds, add a 5 at the end of s
