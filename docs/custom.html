CSS vars

To simplify things A LOT and to keep consistency throughout Skelet., we're using CSS var(). If you're not familiar with CSS vars learn more here.

Skelet. var()

On the left is a list with vars you can copy, in your CSS file, and change colors, font-size, paddings, margins etc. throughout Skelet.

Inline var()

You can also use these vars inline directly on your element so you can define styles per element.

A good example are animations. Previously Skelet. used classes to define animation duration and delay, so you were limited to the defined classes.

Now you can define animation duration and delay by yourself by doing so:

<h1 start animation=fadein ease=easeInOutQuad style="--duration: 1s">
<h1 start animation=fadein ease=easeInOutQuad style="--duration: 1s">
