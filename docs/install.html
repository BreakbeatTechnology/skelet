Getting started is super easy.
All You've to do is download and unzip the file, et voilà. You can start doing your thing.


You've 2 files to pay attention to:

  • skelet.html = is the the HTML5 boilerplate
  • css/skelet.css = is the CSS that you must include into your project in order to make it look Fresh af



Side note: I've a pretty reckless and shitty way to update things. So I suggest you to not upgrade existing projects with newer versions of Skēlet, because that will, most likely, break your layout.
