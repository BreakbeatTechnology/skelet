# Misc

Class Purpose
.clearfix
.full-width Make element full width
.full-width-max Make all elements fit their container
.float-right Float elements right
.float-left Float elements left
.inline Display elements inline
.center Center the elements
.txt-left Align text on left
.txt-center Center the text
.txt-right Align text on right
.align-left Align left an element
.align-center Center an element
.align-right Align right an element


# Tooltips

To use Tooltips simply insert your text inside data-tooltip="..." and use one of the data-tooltip-pos below to position it.

<a href="#" data-tooltip="Whats up!" data-tooltip-pos="up">...</a>
Tooltip Code
Tooltip Top data-tooltip-pos="up"
Tooltip Right data-tooltip-pos="right"
Tooltip bottom data-tooltip-pos="bottom"
Tooltip Left data-tooltip-pos="left"
