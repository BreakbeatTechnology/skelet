# Grids

Skēlet. features a 12-column grid system. It’s flexible, easy to use, and fully responsive.



Fluid

Percent based widths allow fluid resizing of columns and rows.

.col-6 {
  flex-basis: 50%;
}

Simple Syntax

All you need to remember is row, column, content.

Containers can be used to wrap sections of your page. Use the .col-cont class to create a responsive, fixed-width container or the .col-cont-fluid class to create a fluid container.

<div class="row">
    <div class="col-12">
        <div>12</div>
    </div>
</div>

Offsets

Offset a column

<div class="row">
    <div class="offset-3 col-9">
        <div>offset</div>
    </div>
</div>

Auto Width

Add any number of auto sizing columns to a row. Let the grid figure it out.

<div class="row">
    <div class="col">
        <div>auto</div>
    </div>
</div>

Nested Grids

Nest grids inside grids inside grids.

<div class="row">
    <div class="col">
        <div>
            <div class="row">
                <div class="col">
                    <div>auto</div>
                </div>
            </div>
        </div>
    </div>
</div>

Responsive Columns

There are five responsive tiers in Skelet: xs, sm, md, lg, and xl. You can use these tiers to change the way the grid responds at various breakpoints. The grid is mobile-first, so the default tier is xs.


Use the col-[sm|md|lg|xl]-* modifier to target a specific tier. Note that tiers are based on minimum widths, so using col-sm-6 will target sm, md, lg, and xl screens. Of course, you can target multiple tiers on the same column as needed.


For example, the following columns will stack on xs screens, take up 50% each (6 out of 12 columns) on sm screens, and 75% and 25% respectively on md, lg, and xl screens.


1st column
2nd column
<div class="row">
<div class="col-12 col-sm-5 col-md-6">1st column</div>
<div class="col-12 col-sm-6 col-md-5">2nd column</div>
</div>
