# Flex

To apply this component, just add the .flex-it class to a <div> element. This will create the flex container. By default, all flex items will be aligned to the left and equally matched in height and width.

By default, the flex container is displayed as a block element. To apply the behavior of an inline element while still laying out its content according to the flexbox model, use the .flex-it-inline class instead of flex-it.



Alignment

Class Description
.flex-it-center Add this class to center flex items horizontally.
.flex-it-centered Add this class to center flex items horizontally and vertically (middle of the page).
.flex-it-left Add this class to align flex items to the LEFT (start).
.flex-it-right Add this class to align flex items to the RIGHT (end).
.flex-it-top Add this class to align flex items to the top.
.flex-it-middle Add this class to vertically center flex items.
.flex-it-bottom Add this class to align flex items to the bottom.
.flex-it-space-between Add this class to distribute items evenly, with the first item at the beginning and last item at the end of the main axis.
.flex-it-space-around Add this class to distribute items evenly with equal space on both sides of each item.

Direction

These classes define the axis flex items placed on and their direction. By default, items run horizontally from left to right.

Class Description
.flex-it-row-reverse Add this class to lay out flex items from right to left.
.flex-it-column Add this class to lay out flex items as vertical columns.
.flex-it-column-reverse Add this class to lay out flex items from bottom to top.

Wrap

By default, flex items fit themselves into one row. Add the .flex-it-wrap class, so that the items wrap into another row when they no longer fit the viewport. To change the items' direction so that they run from right to left, add the .flex-it-wrap-reverse class. The following classes modify the alignment of wrapping flex items. To force the flex items into one row, just add the .flex-it-nowrap class.

Class Description
.flex-it-wrap-top Add this class to align multirow flex items to the top.
.flex-it-wrap-middle Add this class to vertically center multirow flex items.
.flex-it-wrap-bottom Add this class to align multiline flex items to the bottom.
.flex-it-wrap-space-between Add this class to distribute item rows evenly, with the first row at the top and last row at the bottom of the container.
.flex-it-wrap-space-around Add this class to distribute rows evenly with equal space at the top and bottom of each row.

Item order

By default, flex items are laid out according to the source order. To display a certain item as the first or last one, just add one of these classes.

Class Description
.flex-it-order-first Displays the item as the first one.
.flex-it-order-last Displays the item as the last one.
.flex-it-order-first-small
.flex-it-order-last-small 		Affects device widths of 480px and higher.
.flex-it-order-first-medium
.flex-it-order-last-medium 		Affects device widths of 768px and higher.
.flex-it-order-first-large
.flex-it-order-last-large 		Affects device widths of 960px and higher.
.flex-it-order-first-xlarge
.flex-it-order-last-xlarge 		Affects device widths of 1220px and higher.


Item dimensions

To determine how much space a flex item should take up, add one of the following classes to the item.

Class Description
.flex-it-item-none The box's size is determined by its content.
.flex-it-item-auto The space is allocated considering the item's content.
.flex-it-item-1 The space is allocated solely based on flex.
